## 🎯 WHAT YOU ARE BUILDING

A **personal AI outfit picker web app** called **StyleMate**.

The user answers a few simple questions via a clean UI (NOT a chatbot — just dropdowns/buttons/selectors), and the app uses AI to suggest 3 complete outfit combinations from their wardrobe stored in Supabase.

The interface is **form-style / card-based** — simple, fast, beautiful. The user picks occasion + sees weather auto-loaded, hits a button, and gets 3 visual outfit options using their actual clothing photos.

## 🗄️ SUPABASE DATABASE SCHEMA

### Table: `wardrobe`

```sql
create table wardrobe (
  id uuid default gen_random_uuid() primary key,
  created_at timestamp with time zone default now(),
  name text not null,
  category text not null,   -- 'shirt' | 'tshirt' | 'pants' | 'trousers' | 'jacket' | 'shoes' | 'shorts'
  color text,               -- e.g. "navy blue"
  color_hex text,           -- e.g. "#1a2e4a"
  occasion_tags text[],     -- e.g. ['casual', 'date', 'beach', 'gym']
  weather_tags text[],      -- e.g. ['sunny', 'cold', 'rainy']
  image_url text not null,  -- Supabase Storage public URL
  notes text
);
```

### Table: `user_profile`

```sql
create table user_profile (
  id uuid default gen_random_uuid() primary key,
  skin_tone_hex text,       -- e.g. "#c68642"
  skin_tone_label text      -- e.g. "Medium Warm"
);
```

### Supabase Storage Bucket: `wardrobe-images`
- Create a **public** bucket called `wardrobe-images`
- Each clothing photo is uploaded here; the public URL is saved in `wardrobe.image_url`

## 🌦️ WEATHER INTEGRATION (Auto-loaded, Dublin)

```javascript
async function getDublinWeather() {
  const res = await fetch(
    "https://api.open-meteo.com/v1/forecast?latitude=53.3498&longitude=-6.2603&current=temperature_2m,weather_code&timezone=Europe/Dublin"
  );
  const data = await res.json();
  const code = data.current.weather_code;
  const temp = data.current.temperature_2m;
  const label = getWeatherLabel(code);
  return { temp, label };
}

function getWeatherLabel(code) {
  if (code === 0) return "Sunny";
  if ([1, 2, 3].includes(code)) return "Partly Cloudy";
  if ([45, 48].includes(code)) return "Foggy";
  if ([51, 53, 55].includes(code)) return "Drizzle";
  if ([61, 63, 65].includes(code)) return "Rainy";
  if ([80, 81, 82].includes(code)) return "Showers";
  if ([95, 96, 99].includes(code)) return "Thunderstorm";
  return "Overcast";
}
```

## 🔌 OPENROUTER API CALL

```javascript
async function getOutfitSuggestions(occasion, weather, wardrobe, skinTone) {
  const userMessage = `
Occasion: ${occasion}
Weather: ${weather.label}, ${weather.temp}°C in Dublin
User skin tone: ${skinTone}

Wardrobe:
${JSON.stringify(wardrobe, null, 2)}

Please suggest 3 outfits.
  `;

  const response = await fetch("https://openrouter.ai/api/v1/chat/completions", {
    method: "POST",
    headers: {
      "Authorization": `Bearer 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`,
      "Content-Type": "application/json",
      "HTTP-Referer": "https://stylemate.app",
    },
    body: JSON.stringify({
      model: "anthropic/claude-3.5-sonnet",
      messages: [
        { role: "system", content: SYSTEM_PROMPT },
        { role: "user", content: userMessage }
      ]
    })
  });

  const data = await response.json();
  const raw = data.choices[0].message.content;

  // Strip markdown fences if present
  const clean = raw.replace(/```json|```/g, "").trim();
  return JSON.parse(clean);
}
```
